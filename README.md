# ReSTIR GI

本项目的实现基于NVIDIA的vk raytrace框架，在这基础上使用ReSTIR GI算法实现实时全局光照


Setup
-----

~~~~ 
git clone --recursive --shallow-submodules https://github.com/nvpro-samples/nvpro_core.git
git clone git@github.com:jerry3423/FinalProject.git
~~~~
